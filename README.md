🧠 Handwritten Digit Recognition

This project implements a deep learning-based system for recognizing handwritten digits (0–9) using a trained neural network model. The application processes input images, performs necessary preprocessing, and predicts the corresponding digit with high accuracy.

Check the app here👇
https://huggingface.co/spaces/stanziinkunsal/Handwritten-Digit-Recognition

🚀 Features
Accurate classification of handwritten digits (0–9)
Strong performance on the MNIST dataset with high prediction accuracy
Image preprocessing (resizing, normalization)
Real-time prediction through a web interface
Built using deep learning techniques
📊 Dataset

The model is trained on the MNIST dataset, a widely used benchmark dataset containing 70,000 grayscale images of handwritten digits.

🛠️ Tech Stack
Python
TensorFlow / Keras
NumPy
Pillow (PIL)
Gradio / Streamlit (for deployment)
